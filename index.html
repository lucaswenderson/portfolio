Newsletter Subscription

Está curioso sobre como criar uma inscrição de newsletter eficaz com validação de e-mail em Vue.js? Então, você está no lugar certo! Este repositório, "newsletter-subscription-vue", é um projeto que desenvolvi com muito carinho. Utilizando Vue.js e TailwindCSS, ele apresenta uma interface intuitiva para inscrição em newsletters, garantindo a validação do e-mail fornecido. E o melhor: após uma inscrição bem-sucedida, os usuários são recebidos com uma página de agradecimento. Explore o código fonte e descubra todos os segredos por trás dessa implementação!

Manda Trends

Se você está em busca de relatórios de e-mail meticulosamente projetados e cheios de conteúdo inovador, permita-me apresentar o "MandaTrends Email Report"! Este projeto é um relatório que enviamos periodicamente aos nossos clientes, destacando as últimas tendências e inovações. O desafio aqui foi transformar um layout fornecido em formato .PSD em um e-mail totalmente responsivo, levando em consideração todas as peculiaridades dos clientes de e-mail. O resultado? Um e-mail visualmente atraente e altamente funcional que se adapta perfeitamente a qualquer dispositivo. O design responsivo foi uma das minhas principais prioridades, e apliquei CSS diretamente nos elementos para garantir uma experiência de visualização perfeita. Explore o projeto, confira a comparação entre o layout original e o resultado final, e inspire-se para seus futuros projetos de design de e-mail!

Formação técnica.

Bacharelado em Ciência da Computação (UFPA)

Durante meu bacharelado na UFPA, mergulhei profundamente nos fundamentos da computação. Aprendi sobre algoritmos, estruturas de dados, sistemas operacionais e muito mais. Esta formação sólida me proporcionou uma base robusta para explorar tecnologias emergentes e desenvolver soluções inovadoras.

HIT (Human Interaction with Technologies Research Group)

No grupo de pesquisa HIT, tive a oportunidade de trabalhar com especialistas em interação humano-computador. Aprendi sobre as melhores práticas para criar interfaces intuitivas, realizar testes de usabilidade e entender as necessidades dos usuários. Esta experiência foi fundamental para aprimorar minhas habilidades em design e desenvolvimento.

HTML, CSS e Javascript

Com uma combinação poderosa de HTML, CSS e Javascript, aprendi a construir páginas web interativas e responsivas. Dominei a arte de criar layouts atraentes, animações suaves e funcionalidades dinâmicas que proporcionam uma excelente experiência ao usuário.

UI e UX Designer

Como designer de UI e UX, desenvolvi uma paixão por criar interfaces de usuário esteticamente agradáveis e funcionais. Aprendi a importância da empatia no design, garantindo que cada elemento seja intuitivo e atenda às necessidades dos usuários. Desde a escolha das cores até a criação de protótipos interativos, cada detalhe é crucial para o sucesso de um projeto.

Java

Java foi uma das primeiras linguagens de programação que aprendi em profundidade. Com sua robustez e portabilidade, pude desenvolver aplicativos versáteis, desde sistemas de back-end até aplicativos móveis. A orientação a objetos e a vasta biblioteca padrão do Java me permitiram criar soluções eficientes e escaláveis.

Angular, Vue e React

Ao explorar os frameworks e bibliotecas mais populares do front-end, adquiri habilidades valiosas em desenvolvimento web moderno. Com Angular, Vue e React, aprendi a construir aplicações SPA (Single Page Application) ricas e dinâmicas, integrando-as com APIs e otimizando a performance. Cada ferramenta ofereceu uma abordagem única, enriquecendo minha experiência como desenvolvedor.

A Magia do Front-end

Optei pelo front-end pois me fascina a junção de design e tecnologia. A capacidade de traduzir ideias visuais em experiências interativas é empolgante e desafiadora. A satisfação de ver um design ganhar vida através do código é incomparável.

Uma cursiosidade é que esse portifolio foi feito completamente com HTML, CSS e JS puros por motivos de desafio pessoal, então deu um trabalinho :)

Desde pequeno, a tecnologia sempre esteve presente em minha vida, graças à influência do meu pai, que trabalha na área. Esse interesse inicial me levou a escolher Ciência da Computação como minha carreira acadêmica, e foi na universidade que minha paixão pelo desenvolvimento de software realmente floresceu.

Como discente de Ciência da Computação, tive a oportunidade de mergulhar profundamente no mundo do desenvolvimento fullstack, com um foco especial no frontend. Durante minha jornada acadêmica, participei de projetos de pesquisa e extensão que me desafiaram a aplicar meus conhecimentos em cenários práticos.

Minhas competências vão desde a criação de identidades visuais até a gestão de projetos com metodologias Agile. Estou sempre em busca de novas tecnologias e ferramentas que possam aprimorar minha capacidade de entregar soluções inovadoras. Estou ansioso para continuar minha trajetória profissional, buscando oportunidades que me desafiem e permitam contribuir significativamente para equipes e projetos inovadores. Se você está em busca de um desenvolvedor apaixonado, comprometido e com uma trajetória diversificada, adoraria conectar-me com você e explorar possíveis colaborações.
